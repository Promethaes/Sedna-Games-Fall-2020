This game REQUIRES a controller!
A to jump
X to attack
B to dash
Y to toggle some UI
LB to interact with ability zones
RT to activate combat ability
Hold LT and use right stick to pick character
Left stick to move, right stick to move camera

----------------------------------------------

To play the game online, you must first open multithreaded.exe (this is the server)
you must open this application through command prompt, and pass in the IP you want to host on as the only argument
In the online menu, type in the IP into the IP text box. You can now use the leaderboard, and make online games!
Everything should be good from there
